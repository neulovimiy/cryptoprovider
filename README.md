# Crypto Provider

Сервис для получения и обработки данных о криптовалютах. Приложение взаимодействует с внешними API и предоставляет структурированные данные для клиентских приложений.

---

## Основные возможности

- **Получение актуальных данных** — курсы и информация о криптовалютах
- **Интеграция с внешними API**
- **Обработка данных** — фильтрация и преобразование ответов
- **API для клиента** — предоставление данных в удобном формате
- **Асинхронная работа** — обработка запросов в реальном времени

---

## Технологии

| Компонент | Технологии |
|-----------|------------|
| Backend | Node.js |
| API | REST / HTTP |
| Работа с данными | JSON |
| Асинхронность | async/await, Promises |

---

## Что реализовано

- Подключение к внешнему криптовалютному API
- Получение и обработка данных
- Серверная логика обработки запросов
- Обработка ошибок и нестабильных API-ответов
- Формирование удобного ответа для клиента

## Цель проекта
