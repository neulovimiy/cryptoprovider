# CryptoProvider

Десктопное приложение для шифрования, дешифрования и цифровой подписи файлов с использованием современных криптографических алгоритмов.

Проект реализует работу с асимметричным и симметричным шифрованием, а также цифровыми подписями на базе библиотеки OpenSSL.

---

## Описание

Приложение предоставляет интерфейс для безопасной работы с файлами:

- шифрование и дешифрование данных
- создание и проверка цифровых подписей
- генерация криптографических ключей

Проект демонстрирует практическое применение криптографии в прикладных системах.

---

## Основные возможности

- Генерация криптографических ключей (RSA, ECDSA, AES)
- Шифрование и дешифрование файлов
- Создание цифровой подписи
- Проверка подлинности и целостности файлов
- Генерация криптографически стойких случайных чисел

## Реализованные алгоритмы

### RSA
- Длина ключа: 2048 бит
- Формат: PEM
- Используется для: шифрования, дешифрования

### ECDSA (Elliptic Curve Digital Signature Algorithm)
- Кривая: secp256r1 (P-256)
- Формат: PEM
- Используется для: цифровой подписи, проверки подписи

### AES
- Длина ключа: 256 бит
- Режим: CBC (Cipher Block Chaining)
- Используется для: симметричного шифрования файлов

### Хэширование
- Алгоритм: SHA-256
- Длина хэша: 256 бит
- Используется для: проверки целостности данных, подписи файлов

## Генерация случайности

Для повышения криптостойкости реализован сбор энтропии на основе:

- движения мыши
- временных меток

Используется для генерации случайных значений (ключей, IV).

## Архитектура проекта

Проект разделён на независимые модули:

| Модуль | Назначение |
|--------|------------|
| rsa_generator | Работа с RSA (ключи, шифрование, дешифрование) |
| ecc_ecdsa | Работа с ECDSA (подпись, проверка) |
| aes_generator | Симметричное шифрование (AES) |
| resource | UI (окна, кнопки, ввод данных) |
| main.cpp | Связка UI и криптографической логики |
